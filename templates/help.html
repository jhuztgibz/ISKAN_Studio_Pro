Enterprise Label Designer

A comprehensive guide to designing, mapping, and printing labels with ISKAN Studio.

Getting Started

ISKAN Studio is a professional label designer. Before designing, set your canvas size in the left sidebar.

Initial Setup Steps:

  • 1.
    Choose Units: Switch between Pixels, Inches, or Centimeters.

    Note: We recommend Pixels for web-only designs and CM/Inches for physical printing accuracy.

  • 2.
    Set Dimensions: Enter the Width and Height of a single sticker.

    Example: For a standard shipping label, use 4x6 inches.

  • 3.
    Visual Aids: Enable "Show Grid" or "Show Ruler" to help align objects.

    The grid snaps every 20 pixels by default.

File & Project Management

Manage your designs as reusable templates or export them for external use.

01
New Design

Clears the current canvas and resets all dimensions to the default 700x500px. A confirmation prompt prevents accidental data loss.

02
Save Template (Template Library)

Saves the current design into the application's Template Library. When you save, provide a name and optional description — the template (objects, positions, styles, and data mapping) is stored for quick access. After saving you can reopen the template for editing or send it directly to printing without downloading a file.

If you need a portable backup, use Export Template to download a .json file.

03
Open Template (My Templates)

Open the My Templates panel from the File menu or the Templates button. Select any saved template to load it into the canvas for further editing or printing. You can also Import Template to restore designs from a previously exported .json file.

04
Save as PNG

Exports the current view of the canvas as a high-resolution image file. If you have a multi-label layout (e.g., 3x7), this will export the first page of that layout.

Pro Tip: Templates saved to the Template Library are available on this machine for quick reuse. Use Export Template to create a JSON backup for sharing or long-term archival.

When collaborating, export and share the JSON file so other users can import and reproduce your design exactly.

3. Inserting Objects

Barcodes & QR

Supports Code128, EAN13, and UPC. Vector-based rendering ensures crisp lines.

  • • Code128: Alphanumeric data.
  • • EAN13: Exactly 12 or 13 digits.
  • • UPC: Exactly 11 or 12 digits.

Text & Labels

Click to select, double-click to edit. Use the sidebar to change font weight, family, and size.

Shapes

Rectangles and Circles. Essential for creating containers, separators, or highlighting key data.

Logos

Upload PNG/JPG images. Use the Fit Design button in the sidebar to auto-scale high-res images to your canvas.

Advanced QR Customization

Our QR generator allows for high-level branding. Select a QR code on your canvas to see these options:

1. Branding with Logos

Upload a logo to place it inside the QR code. You can choose positions (Center, Corners) and enable Rounded Logo for a modern look.

2. Module Shapes

Change the QR "dots" from standard squares to Rounded or Circles. This enhances the aesthetic of luxury or tech labels.

3. Quiet Zones (Padding)

Add padding and change the "Quiet Zone Color" (background behind the logo) to ensure the QR code remains scannable even with a logo overlay.

4. Working with Tables

Tables are perfect for creating structured grids for technical specs or price lists.

Dynamic vs. Static

Use the sidebar to change Rows and Columns instantly. If you need to resize a specific cell, click "Ungroup into Shapes" to edit cells individually.

5. Dynamic Data Binding

The Concept

Design your layout once, then map fields to your data source. ISKAN Studio swaps the values automatically during preview and printing, allowing you to generate thousands of unique labels in seconds.

01
Connect Data

Upload an Excel/CSV or connect a REST API in the Data Source tab.

02
Select Canvas Object

Return to the designer and select any Text, Barcode, or QR Code.

03
Link to Column

In the Properties panel, use the Link to Column dropdown to pick a field from your data.

04
Preview & Print

Click rows in the data table to watch your design update. Use Print Selection to export the results.

Pro Tip: Smart Quantity Control

If your spreadsheet contains a column named Qty, Quantity, or Count, ISKAN Studio will automatically generate that exact number of labels for each record.

REST API Integration

Connect ISKAN Studio directly to your ERP or Warehouse Management System via live API endpoints.

1

Configure Endpoint

Click API Source in the ribbon. Enter your full REST URL (e.g., https://api.myapp.com/products).

2

Authentication & Headers

Provide the Header Name (default is x-api-key or Authorization) and your secure Token.

3

Data Normalization

ISKAN Studio automatically searches the JSON response for common data arrays like data, items, or results. If found, it populates the table instantly.

Security Note

Your API tokens are handled client-side. ISKAN Studio does not store your credentials on any server; they persist only for your active session.

6. Printing & Export

Standard Printing (A4/Letter)

Set "Print Layout" rows and columns (e.g., 3x7) to tile labels on a single sheet. Click "Print" to generate a high-quality PDF.

Thermal Printing (Zebra/TSC)

For direct thermal printing, you must use the Web-Print Connector.

  1. Run the Web-Print Agent on your PC.
  2. Click the Connect button in the top right.
  3. Select your physical printer from the dropdown.
  4. Use the Print Label button to send data directly.

7. Troubleshooting

"Agent: Offline"

Ensure the Python print engine is running. If it is running but still offline, visit https://localhost:8182 in your browser and click "Advanced" -> "Proceed" to trust the local SSL certificate.

Barcode Generation Error

Standard Barcodes (EAN13/UPC) have strict digit requirements. EAN13 must have 12 or 13 digits. If your data doesn't match, the barcode will not render. Use "Code128" for flexible alphanumeric data.

Print Preview looks blurry

This usually happens when your "Canvas Size" is set to a low pixel value. Increase your canvas dimensions or switch to "Inches" to ensure the internal resolution is high enough for sharp printing.
